Usage
1.
Visit http://localhost:3000/scan and scan the QR code with WhatsApp (adjust the URL if needed)
2.
Send messages in the format:
  • in [amount] [description]
  • out [amount] [description]
  • Optional: Add -date:DD-MM-YYYY to specify the date

Examples:

out 10rb lunch -date:23-06-2025 in 300rb project out 15rb snacks

If -date is not provided, today's date will be used automatically.

3.
To show logs, send commands like:
  • -show daily:DD-MM-YYYY
  • -show monthly:MM-YYYY
  • -show yearly:YYYY

Example: -show monthly:06-2025

4.
The bot will reply with a financial summary of your income and expenses.
